## Ships

Look at different ships' data in a table view or add your own.

## Get started

```bash
git clone https://github.com/csengekulker/shipsale
```

```bash
cd shipsale
```

```bash
npm install
```

```bash
npm start
```

> Note that you also need this [REST API](https://github.com/oktat/shipsale) set up and running.
